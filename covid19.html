COVID-19 Analyst

Made using C++, this program sorts and organizes 2020 worldwide COVID-19 data. There are over 50,000 data points in the CSV file in which this program pulls information from. The user has the option to select various countries, a time frame, and whether they'd like to sort by deaths, cases, or rate. User can then can view the top dates with their given selection. This program also records the execution time of sorting the data using two different algorithms: merge sort and quick sort.
